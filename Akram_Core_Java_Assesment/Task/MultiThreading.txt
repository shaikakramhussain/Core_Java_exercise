
		Multi threading concept:
-----------------------------------------------
*)Multitasking allows CPU to perform multiple tasks 
(program, process, task, threads) simultaneously

*)Multithreading allows multiple threads 
of the same process to execute simultaneously.
To save the Cpu ideal time
